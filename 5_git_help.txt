Command:
    git help

Description:
Displays the general Git help menu. It lists commonly used Git commands and how to get more help on each.

Example:
    git help

Output:
    usage: git [--version] [--help] [-C <path>] ...
       ...
       The most commonly used git commands are:
           add        Add file contents to the index
           commit     Record changes to the repository
           status     Show the working tree status
           ...

‚úÖ Use this when you want to explore available Git commands.

-------------------------
Command:
    git <command> --help

Description:
Shows the detailed manual (documentation) for a specific Git command.
Note: it redirects to a website
Examples:
    git status --help
    git commit --help
    git log --help

üîπ Opens a detailed explanation with:
- Command usage
- Flags and options

‚úÖ Press `q` to exit the help page.

3Ô∏è‚É£ git help <command>
-----------------------
Another way to view the manual page for a specific command.

Example:
    git help push

This is the same as:
    git push --help

---

4Ô∏è‚É£ git help --all
-------------------
Command:
    git help --all

Description:
Displays a complete list of all available Git commands ‚Äî not just the commonly used ones.

Useful when you want to explore advanced or lesser-known commands.
